💡 Tips

  • If the program does not load automatically, type J then "" and Enter to load it (J is the ZX Spectrum shortcut for LOAD)
  • If the program does not start automatically after loading, type R then Enter to run it (R is the ZX Spectrum shortcut for RUN)
  • For slow tape loading, disable instant tape loading in the emulator File menu, then type LOAD "", and press the cassette icon in the emulator.
  • If needed, click the screen to capture keyboard input
  • Use the ■ Reset button in the emulator menu if anything goes wrong
